# Features

AutoClaw deploys the full OpenClaw platform with all its capabilities, plus additional managed features that make running your AI assistant effortless.

## OpenClaw Features (Included)

### Chat Channels

Connect to your AI assistant from any messaging platform you already use.

- **20+ channels supported** — WhatsApp, Telegram, Discord, Slack, iMessage, and more
- **Multi-channel simultaneous** — Run multiple channels at once
- **Group chat support** — Mention-based activation in group chats

[See all channels →](/features/channels)

### Skills & Integrations

Extend your agent with powerful integrations.

- **50+ skills available** — GitHub, Gmail, Calendar, Browser, and more
- **OAuth connections** — One-click authorization for supported services
- **Vetted for security** — Every skill reviewed by our team

[See all skills →](/features/skills)

### Voice & Speech

Hands-free interaction with your agent.

- **Voice Wake** — Activate your agent with a wake word
- **Talk Mode** — Continuous voice conversation
- **Text-to-Speech** — Hear responses read aloud

### Browser Control

Let your agent browse the web for you.

- **Automated browsing** — Navigate, click, type, scroll
- **Screenshots** — Visual feedback of web pages
- **Form filling** — Automate web workflows

### Automation

Schedule and trigger agent actions automatically.

- **Cron jobs** — Run tasks on a schedule
- **Webhooks** — Trigger from external events
- **Gmail triggers** — React to incoming emails

### Multi-Platform

Access your agent from any device.

- **WebChat** — Browser-based interface
- **macOS app** — Native companion app with menu bar integration
- **iOS node** — iPhone/iPad companion
- **Android node** — Android companion

No SSH, no server setup, no networking configuration. Click deploy and you're live in minutes.
