## ğŸ Getting Started

### Prerequisites
*   Node.js 24+
*   pnpm

### Running the Website

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/jmoraispk/loglife.git
    cd loglife/website
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    pnpm dev
    ```
    *The site will be available at `http://localhost:3000`.*

4.  **Build for production:**
    ```bash
    pnpm build
    pnpm start
    ```
