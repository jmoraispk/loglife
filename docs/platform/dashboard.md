# Dashboard

The AutoClaw dashboard is your central hub for managing your OpenClaw instance. Access it at [app.autoclaw.io](https://app.autoclaw.io) after signing in.

## Overview

When you open the dashboard, you'll see:

- **Instance status** — Whether your OpenClaw is running, starting, or stopped
- **Quick stats** — Messages processed, API costs, uptime
- **Recent activity** — Latest conversations and events

## Sections

### WebChat

The built-in chat interface for talking to your AI assistant.

- Real-time streaming responses
- Conversation history
- File and image sharing
- Mobile-friendly design

### Skills

Manage your enabled skills and integrations.

- View all available skills
- Enable or disable skills
- Configure OAuth connections
- Set skill permissions

### Channels

Configure messaging platform connections.

- See connected channels
- Add new channels (as they become available)
- View channel health and status

### Settings

Configure your AutoClaw instance.

- **API Keys** — Manage your AI provider credentials
- **Profile** — Update your account information
- **Billing** — View usage and manage subscription
- **Team** — Invite collaborators (coming soon)

### Logs

View logs from your OpenClaw instance.

- Gateway logs
- Agent activity
- Error tracking
- Search and filter

## OpenClaw Control UI

AutoClaw also provides direct access to OpenClaw's native Control UI for advanced configuration:

1. Click **Advanced → OpenClaw UI** in the sidebar
2. This opens the full OpenClaw dashboard
3. Configure channels, agents, sessions, and more

The native OpenClaw UI provides access to all configuration options, including those not yet exposed in the AutoClaw dashboard.

## Mobile Access

The dashboard is fully responsive and works on mobile devices:

- Access from any browser
- WebChat works on mobile
- Manage settings on the go
