## Contact Support

### Email

**support@autoclaw.io**

Best for:
- Account issues
- Billing questions
- Bug reports
- Technical problems

Response time: Within 24 hours (usually faster)

### What to Include

When contacting support, please include:

1. **Your account email** — So we can find your instance
2. **Description of the issue** — What's happening vs. what you expected
3. **Steps to reproduce** — How can we recreate the issue?
4. **Screenshots or logs** — Any relevant error messages
5. **When it started** — Did it work before? What changed?

### Example Support Request

> **Subject:** WebChat not loading responses
>
> **Email:** user@example.com
>
> **Issue:** When I send a message in WebChat, I see the loading indicator but never get a response. This started yesterday.
>
> **Steps:**
> 1. Open WebChat
> 2. Send any message
> 3. Loading spinner appears but never completes
>
> **Logs:** [Screenshot of browser console errors]
>
> **Notes:** It was working fine on Monday. I haven't changed any settings.
