# Twilio WhatsApp Number Setup

---

## Step 1: Buy a Phone Number

- Open Twilio, pick a phone number, and click "Buy."

## Step 2: Create WhatsApp Sender

- In Twilio, open Messaging > Senders > WhatsApp Senders, then click "Create sender."

- Choose a Twilio phone number for WhatsApp and continue.

## Step 3: Connect Your WhatsApp Account

**If already using WhatsApp:**

- Open WhatsApp and change the phone number in Settings to this Twilio number.

**If don't:**

- Create a WhatsApp account on a local number first, then change the number in Settings to the Twilio number.

## Step 4: Verify the Number

- Twilio gets the verification code. Input the code in WhatsApp to complete number verification.

**Where to find the verification code:**

- Open the Twilio console
- Navigate to your phone number or check the messaging logs
- The verification code will appear there

**Important:**

- Make sure you have access to the Twilio console to retrieve the verification code when WhatsApp prompts you.
- The code typically arrives within a few seconds

## Step 5: Log in to WhatsApp Again

Log in to WhatsApp again with the twilio number:

- Option 1: Uninstall WhatsApp, reinstall it, and log in with the same number.
- Option 2: Use another WhatsApp app (e.g., WhatsApp Business) and log in with the same number.
